Derived Higher Order Derivative Formula:

  1. If \( y = x^m\) , then, \(y_n= \frac{m!}{(m-n!)} x^{(m-n)} \) if, m>n, m∈Z+
    • If m=n,
      \(y_n = n!\)
  2. If y = \((ax+b)^m\), then \(y_n = a^n \frac{m!}{(m-n)!} (ax+b)^{m-n}\)
  3. If \(y = e^{ax}\), then \(y_n = e^{ax}.a^n\)
  4. If y = \(\frac{1}{ax+b}\), then \(\frac{-1^n.n!a^n}{(ax+b)^{1+n}}\)
    • If y = \(\frac{1}{x}\), then \(y_n = \frac{-1^n.n!}{(x)^{1+n}}\)
    • If y = \(\frac{1}{x+a}\), then \(y_n = \frac{-1^n.n!}{(x+a)^{1+n}} \)
  5. If y = sin(ax+b), then \(y_n = a^n. sin(n\frac{π}{2}+ax+b)\)
  6. If y = cos(ax+b), \(y_n = a^n cos (n\frac{π}{2}+ax+b)\)
  7. If y = \(e^{ax}sinbx\), then \(y_n = (a^2+b^2)^n.e^{ax}.sin(bx+n.tan^{-1}\frac{b}{a})\)
  8. If y = \(e^{ax}cosbx\), then \(y_n = (a^2+b^2)^{\frac{m}{2}}.e^{ax}.cos(bx+x.tan^{-1}(\frac{b}{a}))\)
  9. If \(y=\frac{1}{x^2-a^2}\), then \(y_n = \frac{-1^n.n!}{2a}[\frac{1}{(x-a)^{1+n}}-\frac{1}{(x+a)^{1+n}}]\)
